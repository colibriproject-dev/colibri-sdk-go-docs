title: Pré-requisitos
---

Para utilizar todo o ferramental disponibilizado pelo Colibri Project, você precisará dos seguintes componentes instalados em seu ambiente:

- **[Go 1.24+](https://go.dev/dl/)**: Linguagem de programação base do *framework*.
- **[Docker](https://docs.docker.com/get-started/get-docker/)**: Essencial para a execução de dependências (banco de dados, cache, etc.) em containers.
- **Make** (opcional): Útil para automatizar tarefas via `Makefile`.
- **IDE de sua preferência**: Recomendamos [GoLand](https://www.jetbrains.com/go/) ou [VS Code](https://code.visualstudio.com/) com a extensão de Go.
