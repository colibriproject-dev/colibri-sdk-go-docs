## colibri-sdk-go-examples

No repositório oficial de [exemplos](https://github.com/colibriproject-dev/colibri-sdk-go-examples), disponibilizamos projetos práticos que demonstram o uso real das funcionalidades do `colibri-sdk-go`.

Neste repositório, você encontrará serviços que exemplificam:
- **Comunicação Síncrona (REST)**: Chamadas entre microsserviços.
- **Comunicação Assíncrona**: Uso de mensageria com AWS, GCP e RabbitMQ.
- **Persistência**: Operações com banco de dados e transações.
- **Observabilidade**: Exemplos de monitoramento e logs.

Explore o código para entender as melhores práticas recomendadas pelo *framework*.
