title: Prerequisites
---

To use all the tools provided by the Colibri Project, you will need the following components installed in your environment:

- **[Go 1.24+](https://go.dev/dl/)**: Base programming language of the framework.
- **[Docker](https://docs.docker.com/get-started/get-docker/)**: Essential for running dependencies (database, cache, etc.) in containers.
- **Make** (optional): Useful for automating tasks via a `Makefile`.
- **IDE of your choice**: We recommend [GoLand](https://www.jetbrains.com/go/) or [VS Code](https://code.visualstudio.com/) with the Go extension.
